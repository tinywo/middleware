#### 二次开发

##### 开发环境
* node: `10.18.0`
* npm: `6.13.6`
        
##### 开发准备
    
1. 克隆本仓库 `git clone git@github.com/tinywo/serialport2websocket.git`
2. 执行 `npm i` 或 `cnpm i` 安装所需依赖包
3. 执行 `npm run start` 启动进行开发调试
4. 执行 `npm run dist` 进行打包
